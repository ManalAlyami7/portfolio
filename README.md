## ğŸ”§ Customization

### Colors
Modify CSS variables in `styles/global.css`:
```css
:root {
    --primary: #2e5939;
    --accent-bright: #3b82f6;
    --bg-primary: #f8fafc;
}
```

### Fonts
Update font families in `styles/global.css`:
```css
body { font-family: "Inter", sans-serif; }
h1, h2, h3 { font-family: "Space Grotesk", sans-serif; }
```
